# Dictation (TTS) — приложение для диктанта

## Установка

```bash
cd C:\projects\dictation
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Возможности

- Вставка текста или загрузка `.txt`
- Разбиение текста на слова (пунктуация игнорируется)
- Режимы:
  - Автоматический: озвучивание по словам или по предложениям
  - По клику: озвучивание по одному слову на каждое нажатие
- Настройки: скорость, пауза, голос/предпочтительный пол
